# Sistema de Administración de Citas

## Instalación y configuración

Para ejecutar este proyecto es necesario tener instalado:

- Java JDK 26
- IntelliJ IDEA
- Git

Clonar el repositorio:

```bash
git clone https://github.com/CarlosGarcia1016/sistema-citas-java
```

---

## Uso del programa

El programa permite:

- Registrar doctores
- Registrar pacientes
- Crear citas médicas
- Relacionar citas con doctores y pacientes
- Administrar acceso mediante usuarios
